## Projects Overview

### 1. TopsTravel (Holiday Booking System).
The most sophisticated initiative in this group, which is to model a full holiday travel club.
* **Key Features:**
* Website and Company Management: monitors various holiday sites and all their sales.
* Membership System: Manages member logins, authentication and member IDs.
* Booking Logic: Members are allowed to book holidays, add friends (companions) and calculate the overall costs.
* Discount Algorithm: Automatically a 10 percent discount is applied to all 10th visits (hits) to a web site.

### 2. myGame (Game Management)
A project aimed at digital product and management interdependence.
* **Key Features:**
* Performance Tracking: Registers the number of downloads and keeps a leaderboard of high score board with player names.
* **Financial Reporting:** A computation method, that is value-returning, is the calculation of total revenue (calculateEarnings) depending on the cost of downloads.
Manager Link: This here employs a separate Manager class in order to match particular staff to the games.

myHouse (Real Estate Simulator) is an educational game offered by PlatinumGames, which serves to teach students how to manage a virtual house business (simulation).
An approximation of property ownership and property maintenance.
* **Key Features:**
* Ownership History: This history traces the present owner and the number of previous owners of the house.
* Sustainability Logic: Has a Goeing Greener approach to revamp energy ratings and reviews on whether or not to impose green taxes.
* Financial Tracking: Separates the original cost of the purchase and the market cost of the purchase.

## Technical Requirements
* Environment: BlueJ (preferred) or any other standard Java IDE.
* **Java Version:** Java 8 or higher.
* Libraries:`java.util.ArrayList` and `Java.util.Iterator` are used.

## How to Use
1. Open BlueJ Click on a particular project folder (e.g. Tops Travel Club ) and then open the package.bluej file.
2. Create Objects: To create a new object instance, right-Click on the class (e.g., on an object class such as Member or Web site).
3. Interact: Click the object created on the object bench, the object has its own methods such as, memberLogin, download and sell, etc.
